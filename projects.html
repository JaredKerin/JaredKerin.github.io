Tsiram Virtual 6502 Processor

Computer Organization and Architecture Project

Built a virtual 6502 processor as part of a Computer Organization and Architecture course. The processor was implemented in TypeScript and executed on Node.js, designed around a fully object-oriented programming structure for modularity and maintainability. The system accepts 6502 op-code programs and runs them through a simulated CPU pipeline, including fetch, decode, execute, write-back, and interrupt handling stages. This project deepened my understanding of low-level processor design, instruction sets, and the relationship between hardware architecture and software.

Tools: TypeScript, Node.js, 6502 Instruction Set
