Machine Learining Stock Predictor

Flask application to predict S&P 500 stock movements using historical data fetched with yfinance and a RandomForestClassifier model. The application processes the data, trains the model, and performs backtesting to evaluate predictions. An HTML template was created to display the precision score, prediction counts, and target percentages.
