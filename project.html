Project 1: DASH: RAG application

AI-powered chatbot leverages Retrieval-Augmented Generation (RAG) to ingest and process training documents, enabling seasonal drivers and PVDs to quickly access information by asking natural-language questions. Built with ReactJS on the frontend and a TypeScript backend, the application integrates Vertex AI for document indexing and GPT-3.5 Turbo for generating accurate responses. Fully containerized with Docker and deployed on Google Cloud, the system eliminates the need to navigate lengthy manuals or disconnected repositories, providing employees with fast, on-demand support through an intuitive interface. Project placed top 10 in 85+ team hackathon

Project 2: Machine Learining Stock Predictor

Project 3: Tsiram virtual 6502 processor

Project 4: Compiler-Inator

Typescript based compiler with a UI for logging. Utilizes a lexer, recursive decent parser and semantic analyser. Logs the Concrete Syntax Tree and Abstract Syntax Tree. Generates 6502a code to be ran in an operating system. Grammer supports addition, IF statements, while loops and more.
